 # More Project Details
 ## Desktop Wind Tunnel Simulation Creation & Physical Model Construction
 ### Notes
 - Tunnel size will depend on the size fan bought and used within the tunnel, still should be relatively small; easy to rescale in SolidWorks
 - Tunnel must have a way to easily insert and remove test models
 - Tunnel ideally should be disassemble friendly, possibly achieved by using flanged connections for each part with possible bolts? Still working on this.
 - At first the goal is simply qualitative analysis, thus a form of smoke or mist visualization needs to be figured out.
 - Qualitative flow measurement is a future, but not primary goal
 - Desire to add in controllability and comparability in both the simulation ans the physical build by introducing variations of the flow straightener (ex: honeycomb shape, grid shape, circles, etc); goal is to see and affirm how this would affect flow velocity and the laminar nature of the flow
